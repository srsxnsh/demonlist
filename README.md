# GDDL Integrated Demonlist

___

Uses the GDDL API to get submission info and displays it as a list on the site.
All API requests are sent through a cloudflare server, it may take up to a few seconds to load the information.
____

## How to make your own:
#### Fork
- **Fork this repo:** the fork needs to be named '**demonlist**' or the config might break.
- If the fork is named otherwise, you will have to edit the name in **line 3** of **getdemons.js** to fit your repo.
#### Edit conf.json:
- **userid**: Your GDDL userid. This should be a number.
- **username**: This can be anything you want.
- **fontURL**: The google fonts URL for your chosen font.
- **fontname**: The name of your chosen font.
#### Deploy
- **Deploy** site with GitHub Pages, or another service.
